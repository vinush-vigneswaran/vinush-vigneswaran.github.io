Deep Learning: Theory and Applications
24th of March 2021

Purpose

The aim of this piece of writing is to summarise important concepts covered in deep learning. A lot of engineers I have worked with implement deep learning models as a "black box" or more fittingly a "magic box", where the purpose is simply to change the hyperparameters and obtain a sufficiently "good" accuracy. Unfortunately, this method does work (to some extent). The real issue is software engineers do not understand deep learning models enough to realise when not to use it. In order to make some real strides in general AI and AI research, we need to understand why deep learning works and develop an intuition for the underlying mathematics.

This piece of writing will attempt to do that.

The main source of information is from the "Deep Learning" textbook by Ian Goodfellow, Yoshua Bengio and Aaron Courville. My reference for the the code and code explanation comes from the official TensorFlow documentation.
