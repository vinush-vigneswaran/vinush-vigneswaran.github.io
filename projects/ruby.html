Ruby v1 : Virtual Assistant | Software
November 2016 | Posted on December 2020

Version 1

A virtual assistant application which integrates Google's cloud speech-to-text to recognise user's command and text-to-speech API to generate a virtual voice. The GUI was created using Tkinter and Rainmeter (for wallpaper visuals). This system was integrated with Arduino for home automation, such that the lights and motorised blinds were controlled by voice.
