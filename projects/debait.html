About the project

The Debait windows app is a conversational AI designed to debate about an article/topic of your choice.
  • Given an article, the user and the AI can converse about the article in the form of a debate.
  • Each statement from the user or AI is classed into 5 categories: Agree, Partially Agree, Disagree, Curious and Statement.
  • Debait uses Co:here's Generate and Classify models via its API.
  • With a bit of prompt engineering, history modelling and classification we can obtain a very powerful output from Co:here's model!
  • GUI designed using Tkinkter.
  • This project was built as part of the Co:here AI 48-hour Hackathon #3
